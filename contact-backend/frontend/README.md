# SwingHub-Inspired One Page Template

A modern, responsive, animation-rich one-page web template inspired by SwingHub. Built using pure HTML, CSS (with responsive media queries & animations), and vanilla JavaScript.

## Features

- Semantic HTML structure
- Responsive layout (desktop / tablet / mobile)
- Playful CSS animations ("cuckoo", blob parallax, hover effects)
- Clean minimalist UI inspired by premium agency websites
- Mobile-friendly navigation menu
- Scroll-reveal animations using IntersectionObserver
- Micro-interaction on contact form
- Zero external dependencies

## Sections Included

- **Header / Navigation**
- **Hero Section**
- **Services Section**
- **Portfolio Section**
- **About and Contact Section**
- **Footer**

## Technologies Used

| Technology | Purpose                          |
| ---------- | -------------------------------- |
| HTML5      | Semantic structure               |
| CSS3       | Responsive design + animations   |
| JavaScript | Interactivity & micro animations |
